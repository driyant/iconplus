# IconPlus Meeting Room Booking

A **Meeting Room Booking** application built with **React**, **TypeScript**, and **Chakra UI**.  
Developed using **Vite** for fast, modern, and maintainable development.

---

## 🚀 Key Features

- **Complete Booking Form** - Select unit, room, date, time, number of participants, and catering
- **Auto Validation** - Real-time validation for participants count, time, and catering
- **Auto-Select Catering** - Catering type automatically selected based on meeting time
- **Modern & Responsive UI** - Built with Chakra UI for professional look
- **Routing** - Using React Router
- **Testing** - With Vitest & Testing Library
- **Mock API** - Master data fetched from JSON files for development & testing

---

## 📁 Project Structure

```text
src/
├── components/      # UI Components (Sidebar, Topbar, Layout, etc)
├── pages/           # Main pages (Home, BookMeeting)
├── router/          # Application routing configuration
├── utils/           # Helpers and global constants
├── theme.ts         # Custom Chakra UI theme
└── main.tsx         # Application entry point
public/
└── mockApi/         # Mock data for development and testing
```

## 🛠️ Getting started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Mode development**

   ```bash
   npm run dev
   ```

3. **Build production**

   ```bash
   npm run build
   ```

4. **Preview build production**

   ```bash
   npm run preview
   ```

5. **Running unit testing**
   ```bash
   npm run test
   npm run test:coverage
   ```
